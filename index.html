So what's about bootstrap?

Bootstrap is an open source CSS framework created at Twitter in mid 10th and was originally named “Twitter Blueprint” and was used as internal tool in the company, but the devs saw much more potential in it and started to develop the project further. So after a few month they presented it as an open source project called Bootstrap on August 19, 2011 and the project quickly became popular
